How CADBOXX Works

  1. Convert any 3D file (STEP, STL, OBJ, FBX etc) to GLB
  2. Assemble parts in the CADBOXX environment with constraint tools
  3. Create animations or run physics simulations
  4. Export animations or simulation results; integrate into your CAD toolchain
  5. Pay or redeem discounts using CADX tokens
