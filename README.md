This is the frontend of the cadboxx project.

CADBOXX is about to change things in the CAD/CAM Design sector by offering what designers have been looking for for years. CADBOXX is a universal 3D Assembly Platform that allows 3D models created in any 3D Software to be assembled easily without buying specialist convertors or uploaders
